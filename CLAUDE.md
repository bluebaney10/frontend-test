# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Fashion e-commerce landing page for the brand **"PLAIMANAS"**. The project is being built from a Figma design mockup (`Web page.jpg` and Figma link: `https://www.figma.com/design/ku9ERu3OAY4kXmahJqzRZ7/Frontend-test?node-id=1-2`).

**Status:** Pre-development — no framework or code scaffolded yet.

## Design Sections (top to bottom)

1. **Navbar** — Logo "PLAIMANAS", nav links (WOMEN, COLLECTIONS, EDITORIAL, NEWS, STORIES), language selector (TH), SIGN IN, cart icon
2. **Hero** — Full-width dark fashion photography banner with brand name overlay
3. **New Arrival** — Centered product image with "NEW" overlay text + description
4. **Bestseller** — Full-width banner image with "BESTSELLER" text + description
5. **Collections Grid** — Three cards: Collection 01 (wide), Collection 02 & 03 (side by side)
6. **Marquee Ticker** — Horizontal scrolling text: "NEW ARRIVAL", "BEST SELLING" with arrow navigation
7. **Features Bar** — Three columns: Worldwide Shipping, Customer Service, Secure Payment
8. **FAQ** — Tabbed categories (Orders, Shipping, Returns & Exchanges, Sizing & Products) with accordion items
9. **Inquiry Form** — Contact form with gender toggle (Female/Hybrid), name, email, phone, subject select, terms checkbox
10. **Footer** — Link columns (About, Help, Social, Legal) + Newsletter email subscribe

## Figma MCP

The `.claude/settings.local.json` grants access to Figma MCP tools for reading design context. The Figma file is owned by the FTL team — ensure the file is shared with the authenticated Figma account before using MCP tools.
